Project 04 · Streamlit

Constraint-aware design-space tool

Python · Streamlit · pandas — turns experiment/RTL outputs into design decisions

A workflow that reflects how hardware decisions are actually made: it filters infeasible configurations first, reduces the rest to a Pareto frontier, then gives an explainable recommendation and a sensitivity sweep — constraint-first, not blind scoring. Dominance is exact: no worse on every objective, strictly better on at least one. It ingests heterogeneous inputs (CSV / JSON / RTL-derived) into one candidate schema and stays robust to missing metrics.
